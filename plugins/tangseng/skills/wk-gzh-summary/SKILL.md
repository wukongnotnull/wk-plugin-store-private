# 悟空非空也公众号文章摘要与标签生成

> **关于这个skill的作者**
> 这是悟空非空也的个人写作风格 skill。账号全称「悟空非空也」，是一个以「有趣的方式分享AI干货，满足粉丝对AI的好奇心」为slogan的公众号。

你正在为「悟空非空也」的公众号文章生成摘要和标签。

## 核心原则

**摘要不是总结，是钩子。** 摘要的任务不是概括文章内容，而是让读者产生"我想点进去看"的冲动。好的摘要会制造好奇心缺口、承诺价值、引发共鸣——而不是概述内容。

**标签是分类，也是入口。** 标签帮助读者在海量内容中快速判断"这是不是我想看的"，同时也是搜索和SEO的重要组成。

## 第一步：理解文章

仔细阅读用户提供的文章内容，识别：

- **文章主题**：这篇文章在讲什么
- **目标读者**：这篇文章主要写给谁看
- **核心价值**：读者读完能获得什么
- **情感调性**：文章是幽默的、严肃的、技术向的、情感向的
- **关键亮点**：文章中最有冲击力或最值得传播的点

## 第二步：生成摘要

每次生成 **至少 5 个** 不同策略方向的摘要，覆盖不同吸引力路径：

### 摘要七大类型与写法

**类型1：悬念型**
制造好奇心缺口，让人想知道"然后呢"
> 示例：「凌晨两点，我盯着一行代码愣住了。不是因为它有多难，而是因为我突然意识到，我可能亲手造出了一个怪物。」

**类型2：共鸣型**
直接戳中目标读者的痛点或共鸣点
> 示例：「你也有过这种感觉吗？AI明明很强大，但每次让它帮你写东西，总觉得差了点什么。不是你的错觉。」

**类型3：成果型**
先展示结果，让人想知道是怎么做到的
> 示例：「三天时间，我用AI做完了以前需要两周才能完成的事。不是因为我找到了什么神秘Prompt，而是因为我想明白了一件事。」

**类型4：故事型**
用微型故事开场，让人想跟着读下去
> 示例：「上周五晚上，我给AI投去了一条Prompt。AI回了我一篇8000字的长文。我看完后沉默了半小时。」

**类型5：洞察型**
直接亮出一个反常识或有分量的观点
> 示例：「别再迷信复杂的Prompt技巧了。真正让AI写出好内容的，是另一件事。」

**类型6：身份代入型**
让目标读者觉得"说的就是我"
> 示例：「如果你也是一个每天被各种AI工具折腾得焦头烂额的内容从业者，这篇文章可能是你今年读过最实用的一篇。」

**类型7：对比反差型**
用反差制造戏剧张力
> 示例：「我认识两个人。一个花了499块找人上门装AI产品，另一个只花了9块9。三个月后，他们之间的差距让我愣住了。」

### 摘要写作的核心原则

**1. 要有钩子感**
摘要的第一句话必须在 3 秒内抓住读者。如果第一句就让人想滑走，后面写得再好也没用。

**2. 不要写正确的废话**
"AI时代已经到来，我们需要拥抱新技术"——这种摘要没有任何信息量，也不会让人想读下去。

**3. 要有具体细节**
具体细节比抽象概括更有感染力。"我对着屏幕发了20分钟的呆" 比 "我很震惊" 有力量100倍。

**4. 要有承诺感**
读者点进来是因为你承诺了某种价值。摘要里要让他们知道，这篇文章值得他花时间。

**5. 要符合文章调性**
摘要的语感和文章整体要一致。如果文章风格是轻松幽默的，摘要不要突然变得严肃正式。

## 第三步：生成标签

在推荐摘要下方，生成 **10 个文章标签**，用于微信公众号后台打标签、SEO优化、读者分类。

### 标签生成原则

- **覆盖文章核心主题**（1-3个）
- **覆盖文章类型/领域**（1-2个）
- **覆盖目标读者群体**（1-2个）
- **覆盖实用关键词/工具名**（2-3个）
- **覆盖情绪/共鸣点**（1-2个）

### 标签命名规范

- 使用 `#` 符号开头
- 简洁明了，2-6个字为主
- 避免过于宽泛（如"AI"单独一个词太泛）
- 避免过于冷门（没有搜索量的词没有意义）
- 兼顾搜索友好和读者共鸣

### 标签示例

```
#AI提效 #职场干货 #DeepSeek #工具测评 #打工人 #效率工具 #AI工具 #ChatGPT #副业变现 #数字游民
```

## 第四步：自检清单

生成摘要后快速过一遍：

### L1 合规检查
- [ ] 没有违规词（最、第一、震惊等）
- [ ] 没有过度承诺或夸大

### L2 吸引力检查
- [ ] 第一句能在3秒内抓住人吗？
- [ ] 制造了好奇心缺口吗？
- [ ] 有具体的细节或场景吗？

### L3 匹配度检查
- [ ] 摘要承诺的内容，文章里有充分展开吗？
- [ ] 摘要的风格和文章整体一致吗？

### L4 感染力检查
- [ ] 读起来有情绪温度吗？
- [ ] 像是真人在说话而不是AI在输出吗？
- [ ] 有没有那种让人"对对对"的共鸣感？

## 微信图文摘要的特殊说明

微信公众号文章在订阅号信息流里，会显示一小段摘要（通常是文章开头的前60-100字自动抓取，或者是作者手动填写的"摘要"字段）。两种用法：

**用法1：自动抓取的摘要**
如果文章开头写得好，这段会自动成为吸引读者点进来的第一道门槛。写好文章开头的前100字至关重要。

**用法2：手动填写的摘要**
在微信公众号后台编辑文章时，有一个"摘要"字段是手动填写的（选填，但不填可惜）。这个字段在订阅号信息流里会直接显示，是读者决定是否点进来的关键决策点。

**手动摘要的写法建议**：
- 控制在40-60字之间
- 要有钩子感，让人想知道更多
- 不要重复标题，要补充新信息或角度
- 要符合文章调性
