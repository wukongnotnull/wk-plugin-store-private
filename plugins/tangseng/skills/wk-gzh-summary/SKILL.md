# 悟空非空也公众号文章摘要与标签生成

> **关于这个skill的作者**
> 这是悟空非空也的个人写作风格 skill。账号全称「悟空非空也」，是一个以「有趣的方式分享AI干货，满足粉丝对AI的好奇心」为slogan的公众号。

你正在为「悟空非空也」的公众号文章生成摘要和标签。

## 核心原则

**摘要不是总结，是钩子。** 摘要的任务不是概括文章内容，而是让读者产生"我想点进去看"的冲动。好的摘要会制造好奇心缺口、承诺价值、引发共鸣——而不是概述内容。

**标签是分类，也是入口。** 标签帮助读者在海量内容中快速判断"这是不是我想看的"，同时也是搜索和SEO的重要组成。

## 第一步：理解文章

仔细阅读用户提供的文章内容，识别：

| 维度 | 问题 |
|------|------|
| 主题 | 这篇文章在讲什么 |
| 读者 | 主要写给谁看 |
| 价值 | 读者读完能获得什么 |
| 调性 | 幽默、严肃、技术向、情感向 |
| 亮点 | 最具冲击力或最值得传播的点 |

详细内容：参见 `summary.md` 第一部分

## 第二步：生成摘要

每次生成 **至少 5 个** 不同策略方向的摘要。

**摘要七大类型**：悬念型 / 共鸣型 / 成果型 / 故事型 / 洞察型 / 身份代入型 / 对比反差型

详细内容及示例：参见 `summary.md` 第二部分

## 第三步：生成标签

生成 **10 个文章标签**，覆盖：
- 核心主题（1-3个）
- 类型/领域（1-2个）
- 目标读者（1-2个）
- 工具/关键词（2-3个）
- 情绪共鸣（1-2个）

详细内容：参见 `summary.md` 第三部分

## 第四步：自检清单

生成摘要后快速过一遍四层检查：

| 层级 | 检查点 |
|------|--------|
| L1 合规 | 无违规词、无过度承诺 |
| L2 吸引 | 3秒内抓人、好奇心缺口、具体细节 |
| L3 匹配 | 承诺内容有展开、风格一致 |
| L4 感染 | 有温度、像真人说话、有共鸣感 |

详细内容：参见 `summary.md` 第四部分

## 输出要求

**必须将生成结果写入文件**：`summary.md`（与 SKILL.md 同目录）

每次生成摘要后，使用 Write 工具创建 `summary.md` 文件。

## 完整输出格式

```markdown
## 文章摘要生成

**文章主题**：[一句话描述]
**目标读者**：[主要写给谁]

---

### 摘要列表

1. **[悬念型]**
   「摘要文字」
   吸引力来源：[为什么能抓住人]

2. **[共鸣型]**
   「摘要文字」
   吸引力来源：[为什么能抓住人]

...（共 5 个以上）

---

### 推荐摘要

🥇 **首选**：「推荐摘要1」
🥈 **备选**：「推荐摘要2」

### 文章标签

#标签1 #标签2 #标签3 #标签4 #标签5 #标签6 #标签7 #标签8 #标签9 #标签10
```

## 微信图文摘要说明

**手动摘要写法**：
- 控制在 40-60 字
- 要有钩子感
- 不重复标题，补充新信息
- 符合文章调性

## 与 wk-gzh-writer 的配合

当用户在写完文章后需要生成摘要时，调用 `wk-gzh-summary` skill 生成 5+ 个不同策略方向的摘要供选择。
