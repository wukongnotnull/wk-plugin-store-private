# 五层自检完整检查条目

## L1 硬性规则检查

绝对不能违反的规则，任何一项不通过就必须修复。详见 `ai_word_ban.md`。

## L2 风格一致性检查

逐项给出"是/否"判断。

### L2-1 开头检查
- [ ] 是否从一个具体的、当下的事件/场景切入而不是宏大叙事？
- [ ] 第一句话是否让读者产生"然后呢？"的冲动？
- [ ] 有没有使用任何教科书式开头？
- [ ] 是否符合黄金三秒开头？

### L2-2 节奏与结构检查
- [ ] 是否有长短句交替？（连续3句以上句式长度相近 = 节奏呆板）
- [ ] 是否有一句话独立成段的"断裂"效果？（全文至少出现3次）
- [ ] 偏离主线的段落后面是否有"扣主线句"拉回来？
- [ ] 有没有用疑问句来制造节奏的刹车和转向？
- [ ] 是否避免了不必要的小标题？（靠口语化转场自然衔接）

### L2-3 口语化检查
- [ ] 是否使用了推荐口语化词组？（全文至少出现8-10个不同的口语化表达）详见 `colloquial_phrases.md`
- [ ] 有没有论述中的故意打破？（重复强调，中途打断，省略主语等）
- [ ] 有没有至少一处自嘲或承认不足？
- [ ] 标点是否用来表达情绪？（"。。。" "？？？" "= ="至少出现其中一种）

### L2-4 标点禁令二次确认
- [ ] 全文是否完全没有冒号、破折号和双引号？

**通过标准**：L2-1全部通过，L2-2至少3/5通过，L2-3至少3/4通过，L2-4通过。

## L3 内容质量检查

内容本身的深度和说服力。

### L3-1 观点支撑检查
- [ ] 每个核心观点是否都有具体的人/场景/细节/数据支撑？
- [ ] 有没有空泛的观点只有论断没有例证？

### L3-2 知识输出方式检查
- [ ] 知识点是否以"聊着聊着顺手掏出来"的方式呈现？
- [ ] 有没有出现"下面我来介绍""首先需要了解"这种教科书式科普？
- [ ] 引用是否自然融入论述？

### L3-3 文化升维检查
- [ ] 有没有至少一处从具体事件连接到更大的文化/哲学/历史参照物？
- [ ] 这个连接是否自然？

### L3-4 对立面与同理心检查
- [ ] 在讲核心观点时，是否有对对方立场的理解和承认？
- [ ] 有没有先站到读者的处境里，再给出自己的视角？

### L3-5 文章类型专项检查（按你的文章类型检查）
- 调查实验型 → 有没有"亲自下场"的叙事感？过程是否有层层递进的发现？
- 产品体验型 → 有没有真实的使用场景？有没有跟其他产品的自然对比？
- 现象解读型 → 是否有"观察→好奇→研究→升维"的推进？
- 工具分享型 → 有没有个人故事铺垫？效果展示是否让人"眼前一亮"？
- 方法论分享型 → 每节是否都落到了可执行行动？是否坦诚说明了学习成本和失败点？有没有谦逊铺垫？

### L3-6 逐一展示检查
- [ ] 如果涉及多个产品/案例的比较，是不是用了逐一展示（每个带吐槽点评）而不是一次性罗列？

**通过标准**：L3-1和L3-2必须全部通过，L3-3至L3-6中至少通过相关项。

## L4 活人感终审

以读者视角通读全文，回答：读完感觉是一个真实的人在认真跟我聊，还是一个AI在输出信息？大声朗读，如果像"念报告"，立刻重写那一段。

### L4-1 温度感
- [ ] 文中的情绪表达是体感记忆还是知识性描述？
- [ ] 如果有人物描写，这个人物是否让你"能感觉到他的体温"？
- [ ] 有没有至少一处让你"被戳中"的真实情绪？

### L4-2 独特性
- [ ] 这篇文章是否有"只有悟空非空也才会写出来的角度"？
- [ ] 文章里有没有一个"哇这个角度我怎么没想到"的洞察？

### L4-3 姿态检查
- [ ] 文章的语气是不是"一个有见识的普通人在认真聊一件打动他的事"？
- [ ] 有没有不自觉地滑入了"导师在教学生"或"品牌在做营销"的姿态？
- [ ] 敢不敢骂人、敢不敢承认自己踩过坑？

### L4-4 心流检查
- [ ] 从头到尾读，有没有哪个地方你的注意力断掉了？
- [ ] 有没有哪段读起来特别"顺"但顺得不像真人说话？
- [ ] 段落之间的转场是不是像在"跳"而不是在"流"？

### L4-5 节奏断点检测
- [ ] 有没有故意制造的不完美（未完成的思维、短句断裂、口语化思维过程）？

**通过标准**：整体感觉"这像是真人写的"。

## L5 内容趣味性检查

文章是否足够有趣、能吸引读者一路往下读。

### L5-1 开头钩子检查
- [ ] 开头是否在3秒内制造了好奇心缺口？读者会不会想"然后呢"？

### L5-2 案例趣味性检查
- [ ] 有没有一个让人"哇"的创意案例或意外角度？

### L5-3 吐槽与情绪检查
- [ ] 有没有真实的吐槽、共鸣点或情绪波动？读者会不会觉得"对对对，我也这么想过"？

### L5-4 节奏变化检查
- [ ] 全文句式是否有长短变化？有没有故意的不完美？

### L5-5 升番检查（适用于多案例/多产品对比类文章）
- [ ] 是否用了逐一展示+升番逻辑，而不是一次性罗列？
