# AI味禁用词/标点/模式清单

## L1-1 禁用词

出现以下词汇，必须替换：

| 禁用词 | 替换建议 |
|---|---|
| 说白了 | 坦率的讲、其实就是 |
| 意味着什么？ | 那结果会怎样呢 |
| 这意味着 | 所以呢 |
| 本质上 | 说到底、其实 |
| 换句话说 | 你想想看、也就是说 |
| 不可否认 | 直接删掉，换成正面陈述 |
| 综上所述 | 换成具体的回扣句 |
| 总的来说 | 换成具体的回扣句 |
| 首先...其次...最后 | 用自然的转场词替代 |
| 值得注意的是 | 删掉，直接说 |
| 不难发现 | 删掉，直接说 |
| 某种意义上说 | 删除，太模糊 |
| 客观来说 | 删除，没有意义 |
| 说白了（AI最爱用） | 坦率的讲 |

## L1-2 禁用标点

| 禁用标点 | 替换建议 |
|---|---|
| 冒号"：" | 用逗号替代 |
| 破折号"——" | 用逗号或句号替代 |
| 双引号""/"" | 用「」替代，或直接不加引号 |
| 单引号'' | 用「」替代，或直接不加引号 |

## L1-3 结构性套话

检查是否出现以下模式：

- "让我们来看看..."
- "接下来让我们..."
- "在当今...的时代"
- "随着...的发展"
- 连续使用 bullet point 罗列观点（超过3个就需要改成散文叙述）
- 大段加粗（超过2行的加粗几乎肯定是过度结构化）

## L1-4 工具名检查

确认所有提到的AI工具/产品都使用了具体名称，没有出现以下空泛表述：
- "AI工具"
- "某个模型"
- "相关技术"
- "这个AI"
