# 短视频脚本公式参考

> 短视频的核心是**三秒一爆点，五秒一转折**。用户注意力只有3秒，必须在第一秒就抓住他们。

## 短视频结构公式

### 公式一：痛点 + 解决方案 + 效果 + 行动指令
最常用的干货/工具类视频结构

```
痛点（0-5秒）→ 解决方案（5-20秒）→ 效果（20-50秒）→ 行动指令（最后5-10秒）
```

**示例**：
- 痛点：你是不是也被重复工作折腾得焦头烂额？
- 解决方案：今天教你一个方法，三天干完两周的活
- 效果：用了这个方法，我效率提升了5倍
- 行动指令：关注我，下期更干货

---

### 公式二：场景 + 冲突 + 转折 + 干货
最适合故事型/经历分享类视频

```
场景（0-5秒）→ 冲突（5-20秒）→ 转折（20-45秒）→ 干货（最后10秒）
```

**示例**：
- 场景：我上周被一个9块9的AI给骗了
- 冲突：本来以为占便宜，结果发现是个坑
- 转折：但没想到这个坑让我发现了一个神器
- 干货：所以我总结了3条防骗指南

---

### 公式三：悬念 + 过程 + 揭秘 + 总结
最适合测评/体验/揭秘类视频

```
悬念（0-5秒）→ 过程（5-30秒）→ 揭秘（30-50秒）→ 总结（最后5-10秒）
```

**示例**：
- 悬念：我发现了一个99%的人都不知道的AI功能
- 过程：就是这个按钮，我点了一下午
- 揭秘：它居然可以自动帮我写周报
- 总结：关注我，带你解锁更多AI骚操作

---

### 公式四：情绪共鸣 + 观点 + 互动
最适合观点输出/情感类视频

```
情绪共鸣（0-5秒）→ 观点（5-30秒）→ 互动（最后10-20秒）
```

**示例**：
- 情绪共鸣：有没有人跟我一样，AI用了一堆越用越焦虑
- 观点：其实不是你不行，是方法错了
- 互动：你们AI焦虑吗？评论区聊聊

---

### 公式五：对比反差型
最适合对比/测评类视频

```
我的做法（0-10秒）→ 别人的做法（10-25秒）→ 结果对比（25-45秒）→ 总结（最后10秒）
```

**示例**：
- 我的做法：加班到凌晨2点做方案
- 别人的做法：用AI半小时搞定
- 结果对比：我的被老板说太普通，别人的直接过了
- 总结：AI时代，努力不重要，方法才重要

---

### 公式六：清单罗列型
最适合干货/技巧类视频

```
开头钩子（0-5秒）→ 清单1（5-15秒）→ 清单2（15-25秒）→ 清单3（25-40秒）→ 引导关注（最后5-10秒）
```

**示例**：
- 开头：这个AI功能，90%的人都不知道
- 清单1：自动整理文件
- 清单2：批量重命名
- 清单3：智能分类
- 引导关注：关注我，下期更炸

---

### 公式七：问答型
最适合知识讲解/答疑类视频

```
抛出问题（0-5秒）→ 分析问题（5-25秒）→ 给出答案（25-45秒）→ 延伸思考（最后10秒）
```

**示例**：
- 问题：为什么你用AI总是得不到想要的结果？
- 分析：因为你在问它，不是在引导它
- 答案：要用"追问式工作流"，别接受第一个答案
- 延伸：关注我，下期手把手教你问

---

### 公式八：蹭热点型
最适合追热点/时效性内容

```
热点引入（0-5秒）→ 热点解读（5-20秒）→ 干货输出（20-45秒）→ 引导互动（最后5-10秒）
```

**示例**：
- 热点引入：ChatGPT又更新了？这次有点东西
- 热点解读：新出的这个功能，简直逆天
- 干货输出：我研究了3小时，给你们总结3个核心用法
- 引导互动：你们觉得这个功能怎么样？评论区聊聊

---

### 公式九：连发型（每5秒一个爆点）
最适合快速干货/高密度信息

```
爆点1（0-5秒）→ 爆点2（5-10秒）→ 爆点3（10-15秒）→ 爆点4（15-20秒）→ 爆点5（20-30秒）→ 关注（最后5秒）
```

**示例**：
- 爆点1：AI写文案第一人称
- 爆点2：自动加emoji
- 爆点3：批量处理
- 爆点4：自带爆款标题
- 爆点5：还能检测敏感词
- 关注：关注我，每天解锁一个AI骚操作

---

## 短视频开头（黄金3秒）

**开场方式选择**：

**糗事开场**：
> "我今天，被一个9块9的AI给骗了。"

**荒诞事实开场**：
> "这个世界，终于魔幻到我看不懂的程度了。"

**悬念开场**：
> "我可能，刚刚被AI给夺舍了。"

**痛点开场**：
> "你是不是也被各种AI工具折腾得焦头烂额？"

**热点开场**：
> "最近这个事，全网都在讨论。"

---

## 短视频节奏要求

- **每30秒必须有一个钩子**：问句、悬念、意外、反转
- **台词要短**：每句不超过15个字，说人话
- **画面要直白**：一镜一意，不玩隐喻
- **语速要快**：适合配音1.5倍速或原速
- **情绪要满**：要么震惊、要么好笑、要么共鸣，不要平淡

---

## 短视频禁止

- 禁止慢慢铺垫背景（"大家好，今天我想跟大家聊聊..."）
- 禁止复杂逻辑链条（"首先...其次...最后..."）
- 禁止没有爆点的陈述（"今天给大家分享一个工具..."）
- 禁止废话（每一秒都是钱）

---

## 短视频文案高级写法

### 三段式结构

```
吸睛点（0-3秒）→ 内容主体（3秒-最后5秒）→ 行动指令（最后5秒）
```

**1. 吸睛点（前3秒）**
通过视觉或语言的冲突感，迅速抓住用户下滑的手指。

**2. 内容主体（保持高信息密度）**
通过反转、金句或实用干货来维持留存。不要平铺直叙，每5秒必须有一个钩子。

**3. 行动指令（最后5秒）**
与其等着观众自己点赞，不如通过一个问题或明确引导，促使他们进入评论区。

---

### 场景化叙事

与其枯燥地介绍产品功能，不如描述一个用户可能遇到的**尴尬瞬间**，而你的内容恰好是那个"解药"。

**核心公式**：
```
尴尬场景（0-5秒）→ 共情连接（5-10秒）→ 引出解决方案（10-20秒）→ 展示效果（20-40秒）→ 行动指令（最后5秒）
```

**好的场景化开场**：
> "你有没有过这种经历——半夜两点，对着电脑发呆，不知道明天开会要说什么..."

**坏的干巴巴开场**：
> "今天给大家分享一个会议准备的技巧..."

**场景化开场的3要素**：
1. **具体的时间**：半夜两点、周一早上、截稿前
2. **具体的动作**：对着电脑发呆、反复修改、不知所措
3. **具体的情绪**：焦虑、崩溃、无助

**场景化叙事示例**：

| 内容主题 | 尴尬场景 | 解药方案 |
|---------|---------|---------|
| AI写周报 | "周一早上，对着空白的周报发呆，不知道这周干了啥" | "试试这个AI工作流，5分钟搞定" |
| 文件整理 | "每次要找文件都要翻半天，电脑桌面乱得自己都看不下去" | "今天教你3步整理文件" |
| AI做PPT | "周五下午4点突然要汇报，PPT还没做，心态崩了" | "用这个AI工具，半小时搞定汇报PPT" |

---

### 节奏感技巧

- **缩短长句**：每句不超过15个字，让文字像呼吸一样有节奏
- **多用动词**：让文字自带画面感，"加班到凌晨"比"非常辛苦"更具体
- **制造冲突**：语言要有对立感，"但是"、"反而"、"居然"
- **控制节奏**：问句后停顿，感叹句后加速

---

### 反直觉设计

在文案开头抛出一个违背常识的结论，强迫大脑停下来寻找答案，提升完播率。

**普通问法**：
> "如何变瘦？"

**反直觉问法**：
> "为什么疯狂运动反而让你更胖？"

**普通结论**：
> "这个工具很好用"

**反直觉结论**：
> "这个工具烂到我都想骂人，但用完之后我服了"

---

### 互动留白法

在文案结尾处故意留下一个争议点或未解之谜，把评论区变成用户的"战场"。

**结尾问句模板**：
- "如果是你，你会当面拒绝老板吗？"
- "这种事，你有没有遇到过？"
- "你觉得哪个是对的？"

**争议性结尾**：
> "说了这么多，肯定有人要骂我。但我还是坚持这个观点..."

---

### 爆款拆解工作法

在写脚本之前，先去抖音搜索你所在领域的关键词，筛选出近一周点赞最高的前50条视频。

拆解内容：
1. **开头第一句话**——记录下来，分析为什么能抓住人
2. **核心转折点**——记录转折是怎么设计的
3. **评论区热评**——分析用户为什么被戳中

然后进行**结构模仿**而非内容抄袭：
- 它用了什么开场结构？
- 它的转折点在哪里？
- 它用什么方式引导互动？

用这个框架重新套用到你的内容上。
